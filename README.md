<h2>About Me</h2>
<p>
Hi, I’m Shehan Iroshan, a Software Engineer and Full-Stack Developer. I build scalable backend systems with <strong>Java, Go, Python, Node.js</strong> and modern frontend apps using <strong>React, Next.js, Angular, NestJS, and Spring Boot</strong>.  
I design secure authentication systems (JWT, OAuth2, RBAC), work with <strong>MySQL, PostgreSQL, MongoDB</strong>, and deploy containerized applications using <strong>Docker & Kubernetes</strong>.  
Outside of work, I enjoy creating open-source desktop apps, VS Code extensions, and hobby game projects in <strong>Unity</strong> and <strong>Godot</strong>. Check my GitHub for my latest projects!
</p>
